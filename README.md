- Python 3.11+

### 1. Clone the repo

```bash

```

### 2. Set up the backend

```bash
cd backend

# Create virtual environment
python -m venv venv

# Activate it
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt
```

### 3. Configure environment

```bash
cp .env.example .env
```

Open `.env` and add your keys:

```
# Primary â€” free, fast, generous limits

# Fallback â€” used if Groq fails or quota exhausted
GEMINI_API_KEY=your_gemini_key_here
```

Get your Groq key (recommended) â†’ [console.groq.com](https://console.groq.com)  
Get your Gemini key (fallback) â†’ [aistudio.google.com](https://aistudio.google.com)

The app works with just one key. Groq has much higher free-tier limits than Gemini.

### 4. Start the backend

```bash

```

You should see:
```

```

Health check: open [http://localhost:8000](http://localhost:8000) â€” you'll see `{"status": "running"}`.

### 5. Run the frontend

Open `frontend/index.html` with **Live Server** (VS Code extension) on port 5500.


### 6. Upload and analyze

Drop any PDF, DOCX, or TXT legal document into the upload card. The analysis takes 5â€“15 seconds depending on document size.
